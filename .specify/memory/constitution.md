# Constitution

## Purpose
Build a recruiter-facing professional website showcasing experience, projects, and technical expertise.

## Principles
- Clarity over complexity
- Content-first design
- Mobile responsive
- Fast-loading
- Senior-level presentation

## Non-Goals
- No unnecessary animations
- No over-engineering
- No unrelated experiments

## Technical Standards
- Next.js
- TailwindCSS
- Netlify hosting
